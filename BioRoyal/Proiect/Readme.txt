Site-ul este pentru o firma imaginara numita BioRoyal. Aceasta firma se ocupa cu vanzarea produselor bio si promovarea unei vieti sanatoase.

Pagina Home:
Este o pagina de deschidere unde gasiti cateva informatii. Pagina a fost realizata cu elemente de html si css.

Pagina About Us: 
Este o pagina ce descrie in cateva cuvinte Istrora firmei, parteneri, sursa produselor si valorile firmei. De asemenea aceasta pagina a fost
realizata cu elemente html si css.

Pagina Our Products:
Este o pagina ce descrie in cateva cuvinte si imagini parcursul produselor de la cresterea acestora pana la livrarea produselor la consumator. De asemenea aceasta pagina a fost realizata cu elemente html si css.

Pagina Buy:
Pagina Buy se ocupa cu comercializarea produselor disponibile. Pe langa elementele de html si css apar si elemente de js.
Cu ajutorul javascriptului am adaugat produsele in pagina si am calculat totalul cosului de cumparaturi.
